# 🎬 Disney Filmography App

Application Android native développée avec Jetpack Compose, connectée à Firebase.

## 📱 Fonctionnalités

- **Authentification** : Inscription en 2 étapes et connexion via Firebase Auth
- **Liste des franchises** : Parcourir toutes les franchises Disney (Marvel, Star Wars, Pixar, etc.)
- **Films par franchise** : Affichage des films organisés par sous-saga avec date de sortie et genre
- **Statuts personnalisés** : Marquer un film comme Vu / Veut voir / Possède le DVD / Veut s'en débarrasser
- **Profil utilisateur** : Statistiques personnelles et gestion de la collection
- **Recherche** : Barre de recherche pour trouver rapidement une franchise
- **Favoris partagés** : Voir quels utilisateurs possèdent un film et veulent s'en débarrasser

## 🛠 Technologies utilisées

- **Langage** : Kotlin
- **UI** : Jetpack Compose + Material3
- **Base de données** : Firebase Realtime Database
- **Authentification** : Firebase Authentication
- **Architecture** : MVVM
- **Navigation** : Navigation Compose

## 🚀 Installation

1. Cloner le repository
```bash
git clone https://github.com/USERNAME/DisneyFilmography.git
```
2. Ouvrir dans Android Studio
3. Ajouter le fichier `google-services.json` dans le dossier `app/`
4. Lancer l'app sur un émulateur ou un appareil Android
